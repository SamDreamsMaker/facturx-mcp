Installation en 2 minutes

Compatible Claude Desktop sur Windows et macOS

1. Installer le package

$ npm install -g facturx-mcp

2. Configurer Claude Desktop

Fichier : %APPDATA%\Claude\claude_desktop_config.json

{
  "mcpServers": {
    "facturx": {
      "command": "facturx-mcp",
      "env": {
        "FACTURX_LICENSE_KEY": "FTRX-..."  // optionnel — plan Pro
      }
    }
  }
}

3. Parler à Claude

"Génère une facture pour ma prestation de novembre : 3 jours de conseil à 800€/j HT, TVA 20%, client Acme SA à Paris, IBAN FR76..."
